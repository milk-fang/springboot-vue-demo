# springoot-vue-demo

技术栈：vue+elementplus+springboot+mybatisplus

先搭建数据库，项目中数据库为springboot-vue.数据库文件在文件夹中



前后端需要都在本机启动服务器。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
